## Development
To install and work on the website locally:

```bash
git clone https://github.com/maxeisen/MaxEisen.me.git
cd MaxEisen.me
npm install
```

To run local server in develop mode:

```bash
npm run dev
```

To build and serve website locally (rollup):

```bash
npm run build
npm run start
```
